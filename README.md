# Healthcare Appointments API — AI Interview Challenge

This repository simulates a real-world backend service for a healthcare company.

## Setup

**Requirements**: Java 21, Maven, Docker

```bash
# Start infrastructure
docker-compose up -d

# Run the application
./mvnw spring-boot:run

# Run tests
./mvnw test
```

The API will be available at `http://localhost:8080`.

## Rules

1. You **may** use ChatGPT, Claude, Copilot, Gemini, or any AI tool.
2. When you use AI: state what you asked, explain the suggestion, and validate it before applying.
3. You must be able to explain every line of code you submit.
4. Commit your changes incrementally — one logical change per commit.
